About AccessRoute

A community-driven project to make every city navigable — regardless of mobility or sensory ability.

Our Mission

More than 1 billion people worldwide live with some form of disability. Yet most mapping tools treat accessibility as an afterthought. AccessRoute was built from the ground up to change that — combining real-time community reporting with verified accessible route data to give every person the information they need to move freely and confidently.

We believe that a truly inclusive city isn't a special feature — it's the baseline. AccessRoute helps communities identify gaps and advocates for change at the local level.

How It Works

01

Community Reports

Users report accessibility barriers — broken ramps, blocked paths, missing curb cuts — via the web form or API.

02

Verification

Reports are cross-referenced with existing map data and upvoted by the community before being marked on the live map.

03

Route Scoring

Each route receives a live accessibility score based on surface quality, ramp availability, obstacle clearance, and reported issues.

04

Navigation

Users browse the interactive map, filter by accessibility needs, and navigate via the highest-scored routes.

Built With

Python Flask — Lightweight backend API powering routes, reports, and stats.
Leaflet.js — Open-source interactive maps rendered in the browser.
OpenStreetMap — Free, community-maintained base map tiles.
HTML / CSS / JS — Responsive, accessible frontend with no heavy frameworks.
