## Step-by-Step Guide

### Step 1: Search for Hospitals

Type in the search bar:
```
hospital
```

### Step 2: View Nearby Hospitals

You'll see **3 nearest hospitals** with:
- 🥇 **NEAREST** - Closest to you
- 🥈 **2nd CLOSEST** - Second option
- 🥉 **3rd CLOSEST** - Third option

Click **"🎯 Show Safest Route"** on your chosen hospital

## Route Details Explained

### Distance
- **📏 2.3 km** = Total route length
- Calculated from your location to hospital

### Accessibility Score
- **💯 95/100** = Excellent accessibility
- **80-100** = Safe (Green)
- **60-79** = Moderate (Yellow)
- **0-59** = Risky (Red)

### Road Condition
- **🛣️ good** = Smooth, well-maintained
- **moderate** = Some wear, acceptable
- **bad** = Cracks, potholes, avoid if possible

### Ramps
- **♿ 3** = Number of wheelchair ramps on route
- More ramps = better accessibility

### Obstacles
- **⚠️ 0** = No obstacles (best)
- **1-2** = Minor obstacles (caution)
- **3+** = Many obstacles (risky)
