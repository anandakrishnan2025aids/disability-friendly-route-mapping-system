# Search and Filter Features Guide

## Search Bar Functionality

The search bar at the top of the map page allows you to quickly find routes based on:

### What You Can Search For:

1. **Route Names**
   - Example: "Central Park", "Riverside", "Market"
   
2. **Route Descriptions**
   - Example: "paved", "flat", "scenic"
   
3. **Route Features**
   - Example: "wheelchair ramps", "audio signals", "benches"

### How to Use:

1. **Type in the search box** at the top of the sidebar
2. **Results update automatically** as you type
3. **Clear the search** to see all routes again

### Search Examples:

```
Search: "wheelchair"
→ Shows all routes with wheelchair accessibility features

Search: "flat"
→ Shows routes with flat surfaces

Search: "audio"
→ Shows routes with audio crossing signals

Search: "park"
→ Shows routes in park areas
```

---

## Tag Filter Buttons

Quick filter buttons below the search bar for common accessibility needs:

### Available Filters:

| Button | Icon | Description |
|--------|------|-------------|
| **All** | - | Show all routes (default) |
| **Wheelchair** | ♿ | Routes optimized for wheelchair users |
| **Visual** | 👁 | Routes for visually impaired users |
| **Elderly** | 🧓 | Routes suitable for elderly users |
| **Transit** | 🚌 | Routes near public transit |

### How to Use:

1. **Click any tag button** to filter routes
2. **Active filter is highlighted** in green
3. **Click "All"** to remove filters

---

## Combining Search and Filters

You can use both search and tag filters together:

### Example Combinations:

**Scenario 1: Wheelchair user looking for flat routes**
- Click "♿ Wheelchair" tag
- Type "flat" in search box
- Result: Only wheelchair-accessible flat routes

**Scenario 2: Finding routes with benches**
- Click "🧓 Elderly" tag
- Type "bench" in search box
- Result: Elderly-friendly routes with rest benches

**Scenario 3: Transit-accessible routes near parks**
- Click "🚌 Transit" tag
- Type "park" in search box
- Result: Routes near transit with park access

## Map Layer Filters

Additional filters at the bottom of the search section:

### Filter Options:

| Button | Shows |
|--------|-------|
| **All** | Everything on the map |
| **Routes** | Only route paths |
| **Points** | Only accessibility points (ramps, signals, etc.) |
| **Reports** | Only community-reported issues |

### How to Use:

1. **Click filter button** to toggle visibility
2. **Active filter is highlighted**
3. **Map updates immediately**

## Search Tips

### For Best Results:

✅ **Use specific keywords**
- Good: "wheelchair ramp"
- Better: "ramp"

✅ **Try partial words**
- "wheel" will find "wheelchair"
- "acces" will find "accessible"

✅ **Combine filters**
- Use tag + search for precise results

✅ **Clear filters to start fresh**
- Click "All" tag
- Clear search box

### Common Searches:

```
"accessible"     → All accessible routes
"ramp"          → Routes with ramps
"audio"         → Routes with audio signals
"bench"         → Routes with rest areas
"flat"          → Flat surface routes
"paved"         → Paved routes
"hospital"      → Routes near hospitals
"transit"       → Routes near public transport
"scenic"        → Scenic routes
"short"         → Shorter routes
```

## No Results?

If you see "No routes found":

1. **Check your spelling**
2. **Try broader search terms**
3. **Remove tag filters** (click "All")
4. **Clear the search box**
5. **Try different keywords**

## Examples by User Type

### Wheelchair Users:
1. Click "♿ Wheelchair" tag
2. Search: "ramp" or "flat" or "curb"
3. Look for high accessibility scores (85+)

### Visually Impaired:
1. Click "👁 Visual" tag
2. Search: "audio" or "tactile" or "signal"
3. Check for audio crossing features

### Elderly Users:
1. Click "🧓 Elderly" tag
2. Search: "bench" or "rest" or "gentle"
3. Look for shorter distances

### Transit Users:
1. Click "🚌 Transit" tag
2. Search: "station" or "bus" or "metro"
3. Check proximity to stops

## Troubleshooting

**Problem**: Search not working
**Solution**: Refresh the page (F5)

**Problem**: No results showing
**Solution**: Clear all filters and search

**Problem**: Wrong routes appearing
**Solution**: Check active tag filter

**Problem**: Can't see routes on map
**Solution**: Click "Routes" in layer filters
