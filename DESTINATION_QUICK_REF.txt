🎯 HOW IT WORKS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. TYPE DESTINATION in search bar
   Example: "hospital", "park", "metro"

2. CLICK "Find Accessible Routes" button

3. VIEW 3 ROUTES ranked by safety:
   🥇 SAFEST (Green) - RECOMMENDED
   🥈 Alternative (Yellow) - Backup
   🥉 Option 3 (Red) - Use with caution

4. SELECT SAFEST ROUTE (green button)

5. START VOICE NAVIGATION

🎨 ROUTE COLORS & MEANING
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🟢 GREEN (Solid line)
   Score: 80-100
   Status: SAFE ✅
   Recommendation: USE THIS ROUTE

🟡 YELLOW (Dashed line)
   Score: 60-79
   Status: MODERATE ⚠️
   Recommendation: Use with caution

🔴 RED (Dashed line)
   Score: 0-59
   Status: RISKY ❌
   Recommendation: Avoid if possible

🔍 WHAT MAKES A ROUTE "SAFE"?
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

HIGH SCORE (80-100) ✅
  ✓ Good road condition
  ✓ Multiple wheelchair ramps
  ✓ No obstacles
  ✓ Well-lit paths
  ✓ Gentle slopes
  ✓ Audio signals
  ✓ Rest benches

MEDIUM SCORE (60-79) ⚠️
  ~ Moderate road condition
  ~ Some ramps
  ~ Minor obstacles
  ~ Adequate lighting

LOW SCORE (<60) ❌
  ✗ Poor road condition
  ✗ Few ramps
  ✗ Multiple obstacles
  ✗ Poor lighting
  ✗ Steep slopes


♿ USER TYPE OPTIMIZATION
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

WHEELCHAIR USERS
  Prioritizes: Ramps, flat surfaces, curb cuts
  Avoids: Steep slopes, stairs

VISUALLY IMPAIRED
  Prioritizes: Audio signals, tactile paving
  Avoids: Complex intersections

ELDERLY USERS
  Prioritizes: Rest benches, gentle slopes
  Avoids: Long distances, steep hills

🔧 TROUBLESHOOTING
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Problem: No routes found
Solution: Check spelling, try different destination

Problem: Location not detected
Solution: Allow browser location access

Problem: All routes low scores
Solution: Choose highest available, report issues

Problem: Routes not showing
Solution: Zoom out, refresh page
