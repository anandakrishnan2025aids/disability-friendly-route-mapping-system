# Hospital & Facility Search Guide

## 🏥 How to Find Nearest Hospitals and Routes

### Quick Start

1. **Type "hospital" in the search bar**
2. **See 3 nearest hospitals** with distances
3. **Click "Find Routes to Here"** on any hospital
4. **View 3 different accessible routes** ranked by safety

---

## Step-by-Step Guide

### Step 1: Search for Hospitals

In the search bar at the top, type:
```
hospital
```

The system will:
- ✅ Detect you're looking for hospitals
- ✅ Find your current location
- ✅ Search for nearest hospitals within 10km
- ✅ Display top 3 closest hospitals

### Step 2: View Hospital Information

Each hospital card shows:

```
🥇 NEAREST
City General Hospital
12 Hospital Rd, Central
📏 0.5 km away | 📞 011-2345-6789
✓ Accessible | 24h
[🗺️ Find Routes to Here]
```

**Information Displayed:**
- **Rank**: 🥇 Nearest, 🥈 2nd Closest, 🥉 3rd Closest
- **Name**: Hospital name
- **Address**: Full address
- **Distance**: How far away (in km)
- **Phone**: Contact number
- **Accessibility**: ✓ Accessible or ✗ Not Accessible
- **Hours**: 24h badge if open 24 hours

### Step 3: Select a Hospital

Click the **"🗺️ Find Routes to Here"** button on your preferred hospital.

The system will:
- Calculate 3 different routes
- Score each route for accessibility
- Rank them by safety

### Step 4: View 3 Routes

You'll see 3 routes displayed:

```
🗺️ Routes to City General Hospital
3 accessible routes found

🥇 RECOMMENDED - Main Route
📏 2.3 km | Score: 95/100 (Safe)
🛣️ Road: good | ♿ Ramps: 3 | ⚠️ Obstacles: 0
[✅ Use Safest Route]

🥈 Alternative - Alternate Route
📏 2.6 km | Score: 78/100 (Moderate)
🛣️ Road: moderate | ♿ Ramps: 2 | ⚠️ Obstacles: 1
[🗺️ View This Route]

🥉 Option 3 - Shortest Route
📏 1.9 km | Score: 52/100 (Risky)
🛣️ Road: bad | ♿ Ramps: 1 | ⚠️ Obstacles: 2
[🗺️ View This Route]
```

### Step 5: Select Best Route

- **Recommended**: Click "✅ Use Safest Route" (green route)
- **Alternative**: Click "🗺️ View This Route" on other options

### Step 6: Start Navigation

After selecting a route:
1. Scroll to Voice Navigation bar at bottom
2. Click "▶ Start" button
3. Listen to turn-by-turn instructions

## Understanding the Results

### Hospital Rankings:

**🥇 NEAREST (1st)**
- Closest to your location
- Highlighted with green marker
- Usually best option in emergencies

**🥈 2nd CLOSEST**
- Second nearest option
- Yellow marker on map
- Good alternative if first is busy

**🥉 3rd CLOSEST**
- Third nearest option
- Red marker on map
- Backup option

### Route Rankings:

**🥇 RECOMMENDED (Safest)**
- Highest accessibility score (80-100)
- Green color on map
- Best road conditions
- Most ramps and features
- Fewest obstacles
- **Always choose this for safety**

**🥈 ALTERNATIVE (Moderate)**
- Medium accessibility score (60-79)
- Yellow color on map
- Acceptable conditions
- Some challenges
- Use if main route blocked

**🥉 OPTION 3 (Risky)**
- Lower accessibility score (<60)
- Red color on map
- Poor conditions
- Many obstacles
- **Avoid if possible**

## Accessibility Information

### ✓ Accessible Hospital:
- Wheelchair ramps available
- Accessible toilets
- Wide doorways
- Accessible parking
- Elevators available

### ✗ Not Accessible:
- May have stairs only
- Narrow doorways
- No accessible facilities
- Consider calling ahead

### 24h Badge:
- Open 24 hours a day
- Emergency services available
- No need to check hours

## Advanced Features

### Filter by User Type:

Before searching, set your disability type:
1. Scroll to "Smart Route Recommendation"
2. Select "Disability Type":
   - Wheelchair
   - Visually Impaired
   - Elderly
3. Then search for hospital
4. Routes optimized for your needs

### Adjust Search Radius:

By default, searches within 10km. To find farther:
1. Use "Nearby Help Finder" panel
2. Adjust radius slider
3. Click "Find Near Me"

## Troubleshooting

### No Hospitals Found:
- **Cause**: No hospitals within 10km
- **Solution**: Try different location or increase radius

### Location Not Detected:
- **Cause**: Location permission denied
- **Solution**: Allow location access in browser settings

### Routes Not Calculating:
- **Cause**: Server error or network issue
- **Solution**: Refresh page and try again

### Wrong Location:
- **Cause**: GPS inaccurate
- **Solution**: Click on map to set manual location
